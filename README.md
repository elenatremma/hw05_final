<h2>LifeTUBE PROJECT by Elena Trembovelskaya</h2>
<h3>Социальная сеть для блоггеров</h3>
  <blockquote>
<b>It's BLOGGERs time!</b> 
  </blockquote> 

<i>Описание</i>
<blockquote>
Благодаря этому проекту можно будет публиковать личные посты и фотографии на предложенные темы: книги, путешествия, цветы жизни.  Также можно подписываться на  других авторов и комментировать их посты. 
</blockquote>

<i>Технологии</i>
<blockquote>
Python 3.7 <br> Django 2.2.19
</blockquote>

<i>Запуск проекта в dev-режиме:</i>  
  <ul>
    <li>Клонируйте реппозиторий:</li>
     <blockquote>
    git@github.com:elenatremma/hw05_final.git </blockquote>
    <li>Создайте и активируйте виртуальное окружение:</li> 
    <blockquote>
    python -m venv venv<br> 
    source venv/Scripts/activate 
    </blockquote>  
    <li>Установите зависимости из файла requirements.txt:</li>
    <blockquote>
    pip install -r requirements.txt
    </blockquote>
    <li>Установите и запустите миграции:</li>
    <blockquote>
    python manage.py makemigrations<br> 
    python manage.py migrate
    </blockquote>   
    <li>В папке с файлом manage.py выполните команду:</li>
    <blockquote>
    python manage.py runserver
    </blockquote>
  </ul>
<b>ENJOY LifeTUBE!</b>
